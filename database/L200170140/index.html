Daftar Perintah MySQL berdasarkan DDL dan DML yang berlaku dalam Server Basis Data

A. Perintah DDL

DDL (Data Definition Language) merupakan perintah SQL yang berhubungan dengan pendefinisian suatu struktur database yang digunakan untuk membuat, mengubah dan menghapus struktur dan definisi metadata dari objek-objek database. Beberapa perintah dasar yang termasuk DDL ini antara lain :
No. Perintah Penjelasan Sintaks
1 CREATE Membuat database baru CREATE DATABASE nama_database;
2 DROP Menghapus database DROP DATABASE nama_database;
3 RENAME Mengubah nama tabel RENAME TABLE Buku To Bukuku;
4 ALTER Mengubah struktur tabel yang telah dibuat ALTER TABLE table_name ADD column_name datatype;

B. Perintah DML

DML (Data Manipulation Language) merupakan perintah SQL yang digunakan untuk proses pengolahan isi data di dalam table seperti memasukkan, merubah dan menghapus isi data. dan tidak terkait dengan perubahan struktur dan definisi tipe data dari objek database. Perintah SQL yang termasuk dalam DML antara lain :
No. Perintah Penjelasan Sintaks
1 UPDATE Memperbarui Data UPDATE nama_tabel
2 INSERT Menyisipkan data INSERT into nama_tabel values (‘value1’, ‘value2’,’ …’)
3 SELECT Mengambil data atau menampilkan data dari satu tabel atau beberapa tabel dalam relasi SELECT nama_kolom from nama tabel
4 DELETE Menghapus data dari tabel DELETE nama_kolom FROM nama tabel where kondisi.

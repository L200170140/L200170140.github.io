SISTEM MANAJEMEN DATABASE KELAS B

Perintah DDL

No. Perintah Penjelasan Sintaks
1 CREATE Membuat database baru CREATE DATABASE nama_database;
2 DROP Menghapus database DROP DATABASE nama_database;
3 RENAME Mengubah nama tabel RENAME TABLE Buku To Bukuku;
4 ALTER Mengubah struktur tabel yang telah dibuat ALTER TABLE table_name ADD column_name datatype;

Perintah DML

No. Perintah Penjelasan Sintaks
1 UPDATE Memperbarui Data UPDATE nama_tabel
2 INSERT Menyisipkan data INSERT into nama_tabel values (‘value1’, ‘value2’,’ …’)
3 SELECT Mengambil data atau menampilkan data dari satu tabel atau beberapa tabel dalam relasi SELECT nama_kolom from nama tabel
4 DELETE Menghapus data dari tabel DELETE nama_kolom FROM nama tabel where kondisi.
